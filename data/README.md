### Инструкция по загрузке билетов
Файл "example.json" - пример, как должен выглядеть файл для подготовки к экзамену

Как заполнять поля?
- "id" - порядковый номер билета
- "question" - вопрос
- "answer" - максимально развернутый, полный ответ на вопрос - эталон, к которому нужно стремиться для получения высокого балла

Готовый файл загрузите в data в виде: Название_дисциплины.json`
